Welcome to DeepNePl

Test Dataset

DeepNePl is a web platform to evaluate the performance of deep learning models in classifying actions based on their electroencephalography signature. The data shape is (N,C,T), where N is the number of instances (i.e. labeled actions recorded), C is the number of channels (i.e. electrodes), and T is the number of time steps recorded for each action. Three convolutional neural networks (CNN), as implemented in Braindecode, are considered:
  • Model A: a 2D CNN where a first convolution of spatial (dimension C) filters is followed by two layers of two-dimensional (dimensions C and T) filters. Maxpooling and batch normalization is used between each convolution. Additional details can be found at arXiv:1611.08024.
  • Model B: a 1D CNN where a first convolution of time filters (dimension T) is followed by a convolution of spatial filters (dimension C) and average pooling. Additional details can be found at arXiv:1703.05051.
  • Model C: a deep CNN with four consecutive blocks of convolutions. Each block consists of two-dimensional (dimensions C and T) filters, followed by batch normalization and maxpooling. Additional details can be found at arXiv:1703.05051.
